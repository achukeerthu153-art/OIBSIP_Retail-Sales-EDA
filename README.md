# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset containing 1000 customer transactions. The objective is to identify sales trends, customer purchasing behavior, and product performance using Python.

## Dataset Information

* Total Records: 1000
* Total Features: 10
* Categories:

  * Beauty
  * Clothing
  * Electronics

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Data Cleaning
* Descriptive Statistics
* Customer Demographics Analysis
* Product Category Analysis
* Revenue Analysis
* Monthly Sales Trend Analysis
* Correlation Analysis
* Data Visualization

## Key Findings

| Product Category | Revenue |
| ---------------- | ------- |
| Electronics      | 156,905 |
| Clothing         | 155,580 |
| Beauty           | 143,515 |

### Insights

* Electronics generated the highest revenue.
* Clothing products showed strong sales performance.
* Beauty products generated comparatively lower revenue.
* Customer demographics provide useful business insights.
* Monthly sales trends help in forecasting demand.

## Business Recommendations

* Increase focus on Electronics inventory and promotions.
* Continue strong marketing for Clothing products.
* Offer discounts and bundles for Beauty products.
* Use demographic insights for targeted campaigns.
* Monitor seasonal sales patterns for inventory planning.
