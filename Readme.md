# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset containing 1000 customer transactions. The objective is to identify sales patterns, customer behavior, and product performance to support data-driven business decisions.

## Dataset Information

* Records: 1000
* Features: 10
* Product Categories:

  * Beauty
  * Clothing
  * Electronics

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Data Cleaning
* Descriptive Statistics
* Customer Analysis
* Product Category Analysis
* Revenue Analysis
* Monthly Sales Trend Analysis
* Correlation Analysis
* Data Visualization

## Key Findings

* Electronics generated the highest revenue (₹156,905).
* Clothing generated ₹155,580 in revenue.
* Beauty generated ₹143,515 in revenue.
* Customer demographics provide useful insights for targeted marketing.
* Sales trends can be used for inventory and promotional planning.

## Business Recommendations

* Increase focus on Electronics products.
* Maintain strong marketing for Clothing products.
* Introduce promotional offers for Beauty products.
* Use demographic insights for personalized campaigns.
* Monitor seasonal sales trends for better forecasting.
